Amgen 2023

Topics: AWS Cloud computing, Full Stack, IgFold, Protein Folding, Generative AI

I am currently an intern at Amgen working on four projects.
  • The first project involves creating an Amazon Machine Image for the Molecular modeling team. By constructing the packer and ansible playbook I am learning a lot about AWS and cloud infrastructure and software configuration as I try to add all the necessary add-ons to the AMI.
  • The second project involves more full stack responsibilties to make an internal webapp that allows users to submit arxiv papers, ask questions about the paper, and recieve technically detailed and accurate answers. Doing this I have become proficient with LangChain, a framework for development using language models, OpenAI api, React, Node.js, Flask, and website deployment on AWS.
  • The third task pertains to setting up a pipeline that allows scientists to predict the structure of antibodies. My task was to take state of the start antibody folding scripts and write the necessary shell, python, and dockerfiles to create a scalable pipeline for folding single or multiple antibodies.
  • My final project involves creating a standard test that the company can use to benchmark the performance of protein folding tools existing or new. With this procedure, the company can carefully evaluate newly released protein folding resources to see if they are worth incorporating. This task effectively sets the stage for all downstream molecular simulations done on potential drug targets.

Peptilogics 2022

Topics:Peptide Folding, PyRosetta, Rosetta, Ab Initio Folding

During the summer going into my junior year, I had the privilege of working at Peptilogics, a company focused on peptide-based theraputics.
  • The first project was to take in custom information encoded on a PDB file, and display it in the log of a molecular visualization program called ChimeraX. The code was written in python using the chimerax.core.commands library, and enabled simulation scientists to have a cleaner view of their desired data from an ensemble PDB.
  • The second project also involved adding commands to ChimeraX. My task was to create a command that allows scientists to color PDB residues based on custom attributes besides b-factors like Rosetta Energy Score.
  • The third task was to create a peptide folding pipeline for the company using the ab initio script on Rosetta. I learned how to use the pyRosetta python library, Rosetta suite, python, and bash scripting to make pipelines to fold peptides in batch. To measure our performance, we used the RMSD and pandas, numpy toolkits to compare against industry standards Alphafold and PEPFOLD. My custom parameters and procedure was able to outperform both programs in the use case of peptides, and now is now being used by the company in their drug discovery pipeline.
